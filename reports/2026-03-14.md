## Update Agent

The update agent successfully applied the v0.2.74 → v0.2.76 bump. Key changes documented in `SKILL-typescript.md`:

- **`getSessionInfo()`** — New API to read metadata for a single session by ID (more efficient than `listSessions()` for single-session lookups); returns `SDKSessionInfo | undefined`
- **`forkSession()`** — New API to fork a session into a new branch with fresh UUIDs; supports `upToMessageId` for checkpoint forks and a custom `title`
- **`listSessions()` pagination** — Added `offset` parameter (skip N sessions from start) enabling paginated session browsing
- **`SDKSessionInfo` type changes**:
  - `fileSize` is now optional (`fileSize?: number`) — only populated for local JSONL storage
  - New `tag?: string` field — user-set session tag
  - New `createdAt?: number` field — creation time in milliseconds since epoch
- **Hooks** — Count updated from 21 to 22 hook events
- **Table of Contents** updated to reflect `getSessionInfo()` and `forkSession()`
